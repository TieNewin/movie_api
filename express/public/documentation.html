myFlix

Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister
Request URL Response
Get List of All Movies /movies Return a list of all movies in external database
Get Data of Single Movie /movies/:title Return data on a single movie by title
Get Genre Data /movies/genres/:genre Return data on a single genre by name
Get Director Data /movies/directors/:director Return data on a single director by name
Post New User /users Register new user
Put New User Username /users/:username Update user's username
Post New Movie to User Favorites /users/:username/movies/:favorites Add movie to a user's list of favorites
Delete Movie From User Favorites /users/:username/movies/:favorites Remove a movie from a user's list of favorites
Delete Existing User /users/:username Remove existing user
